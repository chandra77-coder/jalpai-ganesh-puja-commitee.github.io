Jalpai Site Manager

Edit website text. Then download the JSON file and upload it to data/site-manager.json in GitHub.

Simple use

1. Open this page on your Android phone.
2. Tap browser menu ⋮ and choose “Add to Home screen”.
3. Edit text here.
4. Download the JSON file.
5. Upload/replace it in your GitHub repo at data/site-manager.json.
6. Your website will read the new data after the loader script is connected to index.html.
